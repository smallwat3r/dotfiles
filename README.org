* smallwat3r's dotfiles

Personal dotfiles managed with [[https://www.gnu.org/software/stow/][GNU Stow]].

Feel free to reuse any part of this repository, but note that it is tailored to my own workflow and updated frequently. Back up your existing setup before using the provided Makefile.

** Requirements

- [[https://www.gnu.org/software/stow/][GNU Stow]]

** Installation

Clone into your home directory:

#+begin_src shell
git clone https://github.com/smallwat3r/dotfiles.git ~/dotfiles
#+end_src

Stow all dotfiles (the Makefile detects the OS and selects the appropriate targets):

#+begin_src shell
cd ~/dotfiles && make stow
#+end_src

Other available commands:

#+begin_src shell
make unstow     # remove all symlinks
make dry-run    # preview changes without modifying the filesystem
make dconf-load # load dconf settings (Fedora only)
make help       # list available commands
#+end_src

** Structure

These configurations support multiple platforms:

| Directory | Description                                        |
|-----------+----------------------------------------------------|
| ~base~    | Shared configs, always applied (zsh, Emacs, SSH)   |
| ~macos~   | macOS-specific overrides (~_root/~ for system)     |
| ~fedora~  | Fedora-specific overrides (includes dconf)         |

** Emacs

I use Emacs 30.x with [[https://github.com/doomemacs/doomemacs][Doom Emacs]].
Configuration is stored in [[file:base/.doom.d][base/.doom.d]].

*** Installing Emacs

**** macOS

#+begin_src shell
brew tap d12frosted/emacs-plus && brew install emacs-plus@30
#+end_src

**** Fedora

#+begin_src shell
sudo dnf install emacs
#+end_src

*** Installing Doom Emacs

#+begin_src shell
git clone --depth 1 https://github.com/doomemacs/doomemacs ~/.emacs.d
~/.emacs.d/bin/doom install
#+end_src
